# Anifou —  Website

**Anifou** is a digital space blending data, storytelling, and design.  
This repository hosts the live deployment code for [www.anifou.com](https://www.anifou.com).

---

## About

Anifou explores intersections of:
- **Data & Visualization**
- **Environmental storytelling**
- **Interactive art & analysis**

The platform showcases creative data projects such as  
*Life in Plastic*, and other digital narratives  
built with Python, Flask, HTML, and CSS.

---

## Tech Stack

| Layer | Technology |
|-------|-------------|
| Backend | Python (Flask) |
| Frontend | HTML5, CSS3 |
| Server | Gunicorn |
| Deployment | Linux hosting (live at [www.anifou.com](https://www.anifou.com)) |
